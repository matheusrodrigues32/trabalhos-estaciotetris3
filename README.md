# Desafio Tetris Stack - Tema 3

Este projeto implementa a lógica do jogo **Tetris Stack**, onde o jogador organiza peças de Tetris em tempo real. O desafio envolve o uso de **estruturas de dados** como **fila** e **pilha** para gerenciar a ordem das peças, e o sistema de reserva de peças, permitindo ao jogador decidir quais peças jogar e quais guardar para usar depois.

# Objetivos do Projeto

1. **Fila Circular**: Implementação de uma fila circular para organizar as peças que estão prestes a serem jogadas.
2. **Pilha de Peças Reservadas**: Implementação de uma pilha para armazenar peças temporariamente, permitindo que o jogador armazene e recupere peças conforme necessário.
3. **Integração entre Fila e Pilha**: Sistema que permite a troca de peças entre a fila e a pilha, reorganizando as peças de acordo com a estratégia do jogador.
